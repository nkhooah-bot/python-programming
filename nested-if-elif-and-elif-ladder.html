Nested if, elif, and the elif Ladder

Python Control Structures: Advanced Decision Making

1. Introduction

In previous lessons, we learned to use if to check conditions and else to execute code when those conditions are false. Now, we explore more complex control structures to handle multi-layered logic.

2. Nested if

A nested if involves placing one if or else block inside another. This is fully supported in Python and is essential for handling complex decision-making.

Example: Temperature Check

temperature = float(input("Enter the temperature: "))
if temperature == 25: print("Normal")
else:
  if temperature < 25: print("Cold")
  else: print("Hot")

3. The elif Statement

The elif (else if) statement combines else and if into one compact statement. This helps simplify nested structures by providing the same logic with cleaner, more readable code.

4. The elif Ladder

When you have many conditions to check, the if-elif-else ladder is the preferred structure. It checks multiple levels, and only the first matching condition is executed.

marks = int(input("Enter your marks: "))
if marks >= 90: print("Grade: A")
elif marks >= 75: print("Grade: B")
elif marks >= 60: print("Grade: C")
else: print("Fail")
