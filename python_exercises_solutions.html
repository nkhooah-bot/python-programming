Question 1 – Name Tag

Exercise 1 Name Tag using f-string formatting

Create three variables and use f-string formatting to print one complete sentence.

Variables to create: name, age, city

Required output format:
Hello, my name is __, I am __ years old, and I live in __.

Assign your own values to the variables. Use f-string formatting — do not use the + operator or comma-separated print arguments.

✅ Solution
name_tag.py 
# Create variables and assign values
name = "Thabo"
age  = 25
city = "Maseru"

# Print the sentence using f-string formatting
print(f"Hello, my name is {name}, I am {age} years old, and I live in {city}.")
Expected Output
Hello, my name is Thabo, I am 25 years old, and I live in Maseru.
Key point: An f-string starts with f"...". Variables are placed inside curly braces {variable} directly within the string — Python replaces them with their values automatically.

Question 2 – Swap Values

Exercise 2 Swap two values without using a third variable

Assign the values below, then swap them without using a third variable.

Starting values: a = 5   b = 10

Required output format:
a = 10, b = 5

After swapping, a should hold the original value of b, and b should hold the original value of a.

✅ Solution
swap_values.py 
# Starting values
a = 5
b = 10

# Swap using Python's tuple unpacking — no third variable needed
a, b = b, a

print(f"a = {a}, b = {b}")
Expected Output
a = 10, b = 5
How it works: Python evaluates the right-hand side b, a as a tuple (10, 5) first, then assigns the values back — so a gets 10 and b gets 5 in a single step. No temporary variable is needed.

Question 3 – Swap via Arithmetic

Exercise 3 Swap two values using arithmetic only (no third variable)

Swap the values of a and b using only addition and subtraction — no third variable, no tuple unpacking.

Starting values: a = 10   b = 20

Method: Use the arithmetic swap technique:
Step 1: a = a + b
Step 2: b = a - b
Step 3: a = a - b

Print the result as: After swap: a = , b =

✅ Solution
arithmetic_swap.py 
# Initial values
a = 10
b = 20

# Swapping via arithmetic
a = a + b  # a becomes 30
b = a - b  # b becomes 10  (original value of a)
a = a - b  # a becomes 20  (original value of b)

print("After swap: a =", a, ", b =", b)
Expected Output
After swap: a = 20 , b = 10
Step-by-step trace:
Start: a=10, b=20
After step 1 a = a+b: a=30, b=20
After step 2 b = a-b: a=30, b=10 ← b now holds original a
After step 3 a = a-b: a=20, b=10 ← a now holds original b

Question 4 – Product Variables & Stock Value

Exercise 4 Declare product variables and calculate total stock value

Declare variables for three products, each with a name, price, and stock quantity. Then calculate and print the total stock value for each product.

Products to create:
name1 = "Laptop"   price1 = 999.99   stock1 = 12
name2 = "Mouse"    price2 = 25.50    stock2 = 150
name3 = "Keyboard"   price3 = 79.99    stock3 = 45

For each product calculate: total_value = price × stock and print it.

✅ Solution
products.py 
# Declaring variables and assigning values
name1  = "Laptop"
price1 = 999.99
stock1 = 12

name2  = "Mouse"
price2 = 25.5
stock2 = 150

name3  = "Keyboard"
price3 = 79.99
stock3 = 45

# Calculate total stock value for each product
total1 = price1 * stock1
total2 = price2 * stock2
total3 = price3 * stock3

print(f"{name1}: stock value = ${total1:.2f}")
print(f"{name2}: stock value = ${total2:.2f}")
print(f"{name3}: stock value = ${total3:.2f}")
Expected Output
Laptop: stock value = $11999.88 Mouse: stock value = $3825.00 Keyboard: stock value = $3599.55

Question 5 – Formatted Product Table

Exercise 5 Print a neatly aligned product table with 2 decimal prices

Using the same three products from Question 4, print them as a formatted table with aligned columns. Prices must show exactly 2 decimal places and be prefixed with $.

Required table format:

Product    Price     Stock
Laptop     $999.99   12
Mouse      $25.50    150
Keyboard   $79.99    45

Use f-string alignment: <10 for left-align in a 10-character field, <7.2f for price formatting.
✅ Solution
formatted_table.py 
# Product data
name1  = "Laptop"  ;  price1 = 999.99 ;  stock1 = 12
name2  = "Mouse"   ;  price2 =  25.5  ;  stock2 = 150
name3  = "Keyboard";  price3 =  79.99 ;  stock3 = 45

# Print header row
print(f"{'Product':10} {'Price':8} {'Stock'}")

# Print each product row  — price left-aligned in 7-char field, 2 decimals
print(f"{name1:10} ${price1:<7.2f} {stock1}")
print(f"{name2:10} ${price2:<7.2f} {stock2}")
print(f"{name3:10} ${price3:<7.2f} {stock3}")
Expected Output
Product Price Stock Laptop $999.99 12 Mouse $25.50 150 Keyboard $79.99 45
f-string formatting cheatsheet used here:
{value:<10} — left-align value in a field 10 characters wide
{value:<7.2f} — left-align a float in 7 chars, always show 2 decimal places
The $ is placed before the {price} placeholder so it prints as a prefix.
