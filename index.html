Lecture Summary

This lecture introduces the fundamental concept of a computer program — explaining that a program is a step-by-step procedure or set of instructions designed to perform a specific task. Two simple examples are used to illustrate the concept: calculating the area of a rectangle and computing the total bill amount for purchased items.

The core insight is that every program consists of two key elements: data (the inputs, such as numbers or quantities) and operations (the instructions that process that data to achieve a result). Future lectures will build on this foundation by exploring how to store data in variables and how to perform various operations on it.

Key Concepts & Explanations

What is a Program?

Definition

  • A program is a step-by-step procedure for performing a task or solving a problem.
  • It translates human-understandable steps into a language the computer understands.
  • Programs consist of instructions that guide the computer to perform computations.

Example 1 – Area of a Rectangle

📥 Data (Input)
  • Length value
  • Breadth value
⚙️ Operation & Result
  • Multiply length × breadth

Example 2 – Calculating Bill Amount

📥 Data (Input)
  • List of item prices
  • List of quantities purchased
⚙️ Operations & Result
  • For each item: price × quantity
  • Sum all products
  • Print total bill amount

Data and Operations

Every program, no matter how complex, can be broken down into two fundamental components:

1. Data

  • Inputs or information needed for computation
  • Examples: numbers, lists, text strings
  • The raw material a program works with

2. Operations (Instructions)

  • Actions and computations performed on data
  • Examples: add, multiply, loop, compare
  • Transform data into the desired output

Variables

Variables are named containers that store data values inside a program. They give meaningful names to data (e.g., length, prices) and allow a program to hold and manipulate that data throughout its execution.

Example Code & Use Cases

Code Example 1 – Calculate Area of a Rectangle

rectangle_area.py 
# Assign values to variables
length = 15
breadth = 5

# Perform the operation
area = length * breadth

# Display the result
print('Area is', area)
Explanation: Numeric values are assigned to length and breadth. Their product is computed and stored in area, which is then printed. Output: Area is 75

Code Example 2 – Calculate Total Bill Amount

bill_calculator.py 
# Data: lists of prices and quantities
prices   = [10, 7, 5, 8, 6]
quantity = [1,  2, 3, 1, 1]

# Running total starts at zero
total = 0

# Loop through each item and accumulate cost
for i in range(len(prices)):
    total = total + prices[i] * quantity[i]

# Print the final bill
print('Bill Amount', total)
Explanation: Two lists store prices and quantities. A for loop iterates through them, multiplying each pair and adding the result to total. The breakdown: 10×1 + 7×2 + 5×3 + 8×1 + 6×1 = 10 + 14 + 15 + 8 + 6 = 53. Output: Bill Amount 53

2
