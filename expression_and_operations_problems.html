Python Mathematical Programming Challenges

1. Area of a Trapezium

Write a program to calculate the area using two parallel sides (a, b) and the height (h).

Area = 1/2 * (a + b) * h

2. Area of a Circle

Write a program that calculates the area based on a given radius (r). Use math.pi.

Area = π * r²

3. Kilometers to Miles

Write a program to convert a distance from kilometers to miles.

miles = kilometers * 0.621371

4. Calculate Displacement

Write a program using initial velocity (u), final velocity (v), and acceleration (a) to find displacement (d).

d = (v² - u²) / (2 * a)

5. Surface Area of a Cuboid

Write a program using length (l), breadth (b), and height (h) to calculate total surface area.

TSA = 2 * (l*b + b*h + l*h)

6. Quadratic Equation Root

Write a program using coefficients a, b, and c to calculate one root.

x = (-b + √(b² - 4ac)) / 2a
